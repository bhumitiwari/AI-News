In the terminal run:
```bash
   cd server

```

```bash
   GEMINI_API_KEY=your_gemini_api_key_here
```
In the terminal run:
```bash
  node index.js
```

In a new terminal window run:
```bash
   cd client
   npm install
   npm run dev
```
